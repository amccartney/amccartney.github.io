## 🚀 Getting Started

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev

# or open in browser automatically
pnpm dev -- --open
```

Your portfolio will be live at `http://localhost:5173` (´。• ᵕ •。`)

## 🎨 Development

The project uses:
- **Svelte 5** - Elegant reactive components
- **Vite** - Lightning-fast build tool ⚡
- **SvelteKit** - Full-stack framework

## 📦 Building for Production

Create an optimized production build:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```

> Ready to deploy? Check out [SvelteKit adapters](https://svelte.dev/docs/kit/adapters) for your hosting platform! 🚀
